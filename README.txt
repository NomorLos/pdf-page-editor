PDF Editor-lite

使用方法：
1. 双击 PDF工具.exe。
2. 浏览器自动打开后，点击“添加 PDF”选择一个或多个 PDF。
3. PPT 文件：先选择“不合并 / 2合1 / 4合1 / 6合1 / 8合1”，再点击“PPT转PDF并编辑”或“PPT转PDF保存”。
4. Word 文件：点击“Word转PDF并编辑”或“Word转PDF保存”。
5. 在中间区域查看缩略图；可拖拽页面调整顺序。
6. 选中页面后，可左转、右转、删除。
7. 需要按范围拆分时，在输入框填写 1-3,8,10-12 这类范围，再点击“拆分”。
8. 点击“导出 PDF”会弹出保存窗口；如选择已有文件，系统会询问是否覆盖。

说明：
- 程序只做页面级处理，不直接修改 PDF 原有文字。
- 原文件不会被自动覆盖；导出和拆分都会生成新文件。
- 支持中文路径、中文文件名、带空格的路径。
- Word/PPT 转 PDF 会优先使用 Microsoft Office，失败时尝试 LibreOffice。
- PPT 多页合一输出固定为 A4 竖版。
- 如遇到损坏或加密 PDF，界面会显示错误。错误日志在：
  %LOCALAPPDATA%\PDF Editor-lite\error.log
